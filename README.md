# 注意
解压后，如果无法直接打开</br>
要对着.exe右键点击属性</br>
假在常规标签，下面底下看到</br>
"安全: 此文件来自其他计算机, 可能被阻止帮助保护该计算机"</br>
如果有这个就勾选旁边的解除锁定</br>
然后点击确认</br>
然后再打开应用即可
